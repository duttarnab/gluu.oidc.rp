## Supporting Browser

- Chrome
- Opera

## Quick Install

1. Unpack the zip file in release folder to a empty folder
2. Go to Settings --> Extensions of Chrome browser
3. Switch on the `Developer mode`
4. Click on `Load unpacked` button to load the extension
